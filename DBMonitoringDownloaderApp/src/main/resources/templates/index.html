This DB monitoring program was developed to monitor the health of database systems operated by individuals or businesses. This program was developed as a one-man project, and although it currently offers feeble functionality, we promise to add more features continuously in the future. For Oracle DBMS monitoring, you can perform the following tasks:

  • Archive usage monitoring
  • Tablespace usage monitoring
  • OS disk usage monitoring
  • Alert log monitoring

Please refer to GitHub for detailed source code.
